# scripts/

Standalone utility scripts. Not imported as a library — each is run directly.

- `run_parallel.sh` — Parallel benchmark launcher (distributes studies across N workers)
- `shap_analysis.py` — SHAP feature attribution analysis
- `benchmark_heldout.py` — Held-out test-set evaluation
- `analyze_kappa_optimal.py` — Find optimal kappa per study from benchmark results
